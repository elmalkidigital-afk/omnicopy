# **App Name**: OmniCopy AI

## Core Features:

- Multi-Platform Generation: Generate product descriptions optimized for WooCommerce and Shopify.
- Tone Adaptability: Choose the tone (Luxury, Technical, Friendly, Minimalist) for product descriptions.
- SEO Optimization: Automatically generate SEO-optimized meta titles, meta descriptions, and product tags using an AI tool.
- Direct Exports: Generate CSV files for Shopify and JSON files for WooCommerce for bulk import.
- Credit System: Implement a credit-based payment system (Stripe) for API usage.
- Dashboard: Visualize the generations and provide user friendly access to the generated product descriptions

## Style Guidelines:

- Primary color: Deep indigo (#4F46E5) for a professional and reliable feel.
- Background color: Very light gray (#F9FAFB) for a clean and modern look.
- Accent color: Teal (#34D399) to highlight key actions and elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern, neutral look, suitable for headlines or body text
- Use a set of consistent icons from Lucide (lucide.dev).
- Implement a clean and intuitive layout with a clear separation between the input form and the generated output preview.
- Subtle animations, like fade-in effects, for generated content and loading states to enhance the user experience.
